🌤️ Clima – Weather Forecast Web App

Clima is a responsive weather forecast web application built using HTML, Tailwind CSS, and Vanilla JavaScript.
It allows users to search for any city, view current weather details, and check a 5-day forecast using real-time data from the OpenWeather API.

✨ Features

🔍 City-based weather search

📍 Current location weather using browser geolocation

🌡 Temperature unit toggle (Celsius ↔ Fahrenheit)

📅 5-day weather forecast

🕓 Recent search history stored using localStorage

⚠️ Extreme heat alert for high temperatures

🎨 Dynamic background change based on weather condition

📱 Fully responsive UI (mobile, tablet, desktop)

Api Used:https://api.openweathermap.org/data/2.5/weather || https://api.openweathermap.org/data/2.5/forecast

How the App Works

User enters a city name or uses current location

App fetches real-time weather data from OpenWeather

Weather details are rendered dynamically in the UI

The 5-day forecast is displayed in a responsive grid

Recent searches are saved and shown in a dropdown

Alerts and background changes are applied based on conditions
